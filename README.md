# DROPS-COLLAB

Plantilla simple para lanzar "drops" artísticos/musicales. Pensada para que un usuario sin conocimientos técnicos pueda subir carpetas estáticas al repo y publicar en GitHub Pages.

## Estructura recomendada

/ (raíz)
├── index.html
├── style.css
├── /drops
│     ├── /drop01
│     │    ├── index.html
│     │    ├── style.css
│     │    └── assets/
│     └── /drop02
└── /assets
