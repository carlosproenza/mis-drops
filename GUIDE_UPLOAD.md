# Guía rápida para subir al repo y publicar en GitHub Pages

1. Crear repo en GitHub (añade README).
2. Subir el contenido de esta carpeta a la raíz del repo.
3. En GitHub: Settings → Pages → Branch: main, Folder: / (root) → Save.
4. Esperar ~1 min y visitar: https://TU-USUARIO.github.io/NOMBRE-DEL-REPO

Si viste 404: asegúrate de que 'index.html' esté en la raíz del repo y que GitHub Pages esté apuntando a la rama 'main' y carpeta 'root'.
