# Release Drop 03 - Preview

Archivos añadidos:
- /public/release-drop-03.html -> Página demo estática (abrir en navegador o servir con un servidor estático)
- /assets/css/release-hero.css -> Estilos del hero
- /assets/release-drop-03/hero.svg -> Imagen placeholder para el hero
- /src/components/ReleaseHero.jsx -> Componente React opcional (importa el CSS desde la ruta relativa)

Cómo probar:

Reemplazar la portada cuando la tengas en /assets/release-drop-03/hero.svg o .webp y actualizar la ruta en el CSS si fuese necesario.
